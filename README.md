# React + Vite: Dynamic Color Clock

Clock displayed using React, Javascript and Vite.

## Software/Dependices 
* code editor
* Node.js 
* npm 
* Vite

## How to install and Use 
* cd color-clock 
    * npm install 
    * npm install date-fns@2.30.0 

* run application 
    * npm run dev 
    * copy & paste local: 'http' into web browser
